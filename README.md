# Stock Market

Web socket API for a simulated stock market

## Installation

```shell
npm install
```

## Running the app

```shell
# development
npm run start

# watch mode
npm run start:dev

# production mode
npm run start:prod
```

## Test

```shell
# unit tests
npm run test

# e2e tests
npm run test:e2e

# test coverage
npm run test:cov
```

An example web client is provided. Open [example-client.html](./example-client.html) in a browser.
