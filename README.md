# Stock Market

![demo](https://media.giphy.com/media/oTK6bbeSJFEJJIdbv8/giphy.gif)

## Installation

```shell
npm install
```

## Running the app

```shell
# development
npm run start

# watch mode
npm run start:dev

# production mode
npm run start:prod
```

## Test

```shell
# unit tests
npm run test

# e2e tests
npm run test:e2e

# test coverage
npm run test:cov
```
